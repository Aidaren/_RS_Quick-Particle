# Quick-Particle | Version - 0.1.3

## 中文简介/Chinese Language
**这个模块有什么用？   
此模块集成了很多制作特效时常用的功能，并将其模块化！**
### **脚本功能：**
* **创建Weld**
* **为物体添加速度**
* **获取地面坐标**
* **遍历序列帧**
* **快速发射所有粒子**
* **更多功能正处于开发中**
